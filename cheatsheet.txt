- on Eclipse,
  install http://kindle-sdk.s3.amazonaws.com/addon.xml
  (follow https://developer.amazon.com/sdk/fire/setup.html)
- on S2, install Amazon AppStore (Amazon Web site)
- on S2, get AirDroid from Android Market
- on S2, get Web App Tester from Amazon Store
- look here http://www.enscand.com/roller/enscand/entry/adding_the_amazon_app_store
- on eclipse, install the web app sdk (?)
- get the "Web App Tester.sdk" using Air Droid
- start an emulator:

  cd ~/android-sdks/tools
  android create avd --force -n amazon -t 16 -c 2G  => should be 4.1.2 !!!
  emulator -avd amazon &
  => install Amazon Appstore !

  cd ../platform-tools/
  cp ~/Downloads/Web\ App\ Tester.apk Tester.apk
  adb devices
  adb install Tester.apk
  adb shell
  mount -o remount rw
  exit
  cp ~/jaxoapps/jaxogram/amazon.testerurls.json .
  adb push amazon.testerurls.json /mnt/sdcard/amazon.testerurls.json

- on the emulator, use the search icon to open a browser URL field.
  go to http://www.amazon.com/amazonappstoreapp
